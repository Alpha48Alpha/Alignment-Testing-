## Overview

This repository contains **525 prompts** designed, structured, and evaluated across three major domains to deliver measurable value in AI alignment and robustness testing.

> **Measured outcome: +35% improvement in response consistency across evaluations.**

Each prompt is tagged by subdomain, difficulty level (Beginner / Intermediate / Advanced), and key skills tested. Together they form a production-ready benchmark set for evaluating and improving:

- **Reasoning consistency** — does the model reason the same way across paraphrases?
- **Instruction following** — does the model respect format, length, and method constraints?
- **Factual accuracy** — does the model produce verifiable, correct information?
- **Output clarity** — does the model explain, structure, and communicate effectively?
- **Edge-case handling** — how does the model behave at the limits of its knowledge?

## Evaluation Methodology

| Criterion | Description |
|-----------|-------------|
| **Clarity** | Task is specific and unambiguous |
| **Depth** | Elicits substantive, non-trivial responses |
| **Correctness** | Response verifiable against known knowledge |
| **Instruction-following** | Model respects format, length, and method constraints |
| **Consistency** | Stable quality across paraphrases and re-runs |
| **Edge-case handling** | Exposes model behavior at knowledge boundaries |
