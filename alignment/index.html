Evaluation Framework

Framework evaluations/framework.md

Full evaluation methodology: scoring rubric (0–3 per dimension), consistency measurement, and aggregate score formula across Accuracy, Reasoning Quality, Instruction Following, Hallucination Rate, and Clarity.

Metrics evaluations/metrics.md

Definitions and scoring guidance for all five evaluation metrics. Includes the hallucination quality scoring table and the aggregate score formula: sum / (applicable dimensions × 3) × 100%.

Results evaluations/results/README.md

Summary of evaluation runs across ~500+ prompts. Technical domain: ~200 prompts, 68% consistency, 74% accuracy. Educational: ~150 prompts, 71% / 80%. General: ~150 prompts, 72% / 76%. Relative consistency improvement: +35%.

Prompt Sets — Technical Domain

Prompts prompts/technical/reasoning_quality.md

Reasoning quality test sets: logic and mathematics (train speed, triangular numbers), code reasoning (Python functions), and algorithmic complexity (binary vs. linear search). Each set has three rephrasings.

Prompts prompts/technical/hallucination_rate.md

Hallucination detection prompts: person biography (Alan Turing), scientific claims (double-slit experiment), citation fabrication, legal references (Section 230), and plausible-but-false historical claims (Tunguska event).

Prompts prompts/technical/instruction_following.md

Instruction-following prompts testing format constraints (comma-separated output), length constraints (two-sentence explanations), role constraints (code review), and multi-step task ordering.

Prompt Sets — Educational & General Domains

Prompts prompts/educational/accuracy.md

Educational accuracy prompts across science (speed of light, mitochondria), history (WWII end date), and language (grammar correction). Each set includes three rephrasings with verified expected answers.

Prompts prompts/general/clarity.md

Clarity and readability prompts: common sense (why ice floats), analogical reasoning (CPU explanation for a 10-year-old), ambiguity resolution, geography (largest countries), and plain-language machine learning explanation.
