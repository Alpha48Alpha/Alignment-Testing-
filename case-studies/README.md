# Case Studies

Focused analyses of specific evaluation findings. Each case study documents a concrete behavior pattern observed during evaluation, describes the prompts used, and draws conclusions about model behavior.

## Format

Each case study includes:

- **Observation** — What behavior was recorded
- **Prompts used** — The exact or representative prompts that triggered the behavior
- **Model responses** — Representative outputs (paraphrased or verbatim)
- **Analysis** — Why this behavior occurred and what it reveals
- **Takeaway** — Implications for prompt design or model evaluation
