# Benchmarks

This folder contains benchmark prompt sets and their definitions.

## Included Benchmark Categories

| Category | Description |
|---|---|
| Reasoning Tests | Logical and mathematical reasoning prompts |
| Instruction Following Tests | Format, length, and tone constraint prompts |
| Knowledge Accuracy Tests | Factual recall and knowledge reliability prompts |
| Prompt Robustness Tests | Adversarial and injection-resistance prompts |

Add new benchmark files as Markdown (`.md`) or JSON (`.json`) following the existing prompt file format.
