# Defense Strategies for Prompt Attacks

Effective defenses against prompt attacks include:

## Input Validation

Filter malicious instructions before they reach the model.

---

## Instruction Hierarchy

Ensure system prompts cannot be overridden by user prompts.

---

## Output Monitoring

Detect and block responses that reveal confidential instructions.

---

## Prompt Isolation

Separate system instructions from user inputs to prevent manipulation.
