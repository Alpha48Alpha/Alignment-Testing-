## Evaluation Methodology

Each prompt was evaluated across the following dimensions:

| Criterion | Description |
|-----------|-------------|
| **Clarity** | Is the task specific and unambiguous? |
| **Depth** | Does the prompt elicit substantive, non-trivial responses? |
| **Correctness** | Can the response be verified against known knowledge? |
| **Instruction-following** | Does the model respect constraints (format, length, method)? |
| **Consistency** | Does the model give similar quality across paraphrases? |
| **Edge-case handling** | Does the prompt expose model behavior at knowledge boundaries? |

### Difficulty Scale

| Level | Description |
|-------|-------------|
| **Beginner** | Accessible with minimal domain knowledge; clear single-step answer |
| **Intermediate** | Requires domain familiarity; multi-step reasoning; some nuance |
| **Advanced** | Deep expertise required; open-ended; trade-offs or proofs involved |
